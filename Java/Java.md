# JAVA

- 다양한 환경에서 사용가능 JVM을 통해
- 컴파일을 진행
- 객체 지향 언어

## JVM

- Java Virtual Machine
- OS에 종속되지 않음
- Java compiler를 통해 `*.java` 파일을 `*.class` 파일로 변환
- 어떻게 컴파일 하는가

```bash
>javac test.java
>java test.class
```
